常见问题（FAQ）

什么是 KV Cache 分层存储加速？

把推理中占用显存的 Key/Value 张量按热度分层到外置全闪（NVMe-oF over RoCEv2），按需流式调回。480B 生产部署形态签字级实测：吞吐 +29–40%、TTFT ↓26–32%（R2/R3）。

为什么是优化存储而不是继续加 GPU？

长上下文冷恢复要么重算 prefill、要么读回 KV，两者都在让 GPU 空等。R2 实测：外置 KV 分层对无外存重算加速 8.6–20×，通常比继续堆卡更经济。

实测数据可信吗、如何复现？

全部数字出自签字级/正式版测试报告 R1–R9（证据页 https://mingxinstorage.xyz/evidence）；R8 为代码/数据导出包（LMCache 补丁、负载客户端、编排与取证脚本、原始数据），第三方可独立复现全部结论。

铭信支持哪些国产/非 N 卡算力平台？

R1–R4 主实测平台为 AMD MI308X ×8（ROCm 7.2 + vLLM 0.20.1+rocm721）；R9 在华为 Atlas 910B 昇腾平台实测模型加载 6.2–9.3× vs NFS（平台口径如实标注）；R5 覆盖沐曦 N260。
